# Equity Link - Frontend

## **Framework:** ReactJS

## Requisitos

- Node.js ≥ 18
- NPM ≥ 9

---

## Instalación y ejecución

1. Instalar dependencias:

```bash
npm install
```

2. Iniciar servidor de desarrollo:

```bash
npm start
```

3. Acceder a la app en `http://localhost:3000`.
